# SerenePath Clinic Website - Technical Requirement Document (TRD)

## 1. Objective

Build a modern, calming, interactive website for a psychology clinic
with glassmorphism UI and smooth animations.

## 2. Pages

-   Home
-   Services
-   About
-   Blog
-   Contact

## 3. Design System

### Colors

-   Primary Green: #5E7D6A
-   Soft Green: #A7BFB2
-   Background: #F5F5F3
-   Beige Card: #E8D5C4
-   Text Dark: #2B2B2B
-   Text Muted: #6F6F6F

### Typography

-   Headings: Serif (Playfair / Georgia)
-   Body: Inter

## 4. Core Components

### Navbar

-   Sticky
-   Transparent to solid on scroll
-   CTA: Book Now

### Hero Section

-   Left text, right organic image
-   CTA buttons
-   Glassmorphism overlay

### Services Section

-   Grid layout (2x2)
-   Card types: text, image, colored
-   Hover: scale + shadow

### CTA Banner

-   Background image with overlay
-   Tag chips
-   CTA button

### Testimonials

-   Centered quote
-   Italic serif typography

### Contact

-   Form (Name, Email, Message)
-   Info cards
-   Map embed

### Blog

-   Featured article
-   Grid layout

### About

-   Doctor intro
-   Timeline
-   Certifications

## 5. Animations

-   Library: Framer Motion
-   Fade + slide animations
-   Hover scale
-   Scroll reveal

## 6. Tech Stack

-   Next.js
-   Tailwind CSS
-   Framer Motion
-   React Hook Form

## 7. Folder Structure

app/ components/ lib/

## 8. Performance

-   Lazy loading
-   Optimized images

## 9. Accessibility

-   Keyboard navigation
-   ARIA labels

## 10. Notes

-   Keep animations subtle
-   Maintain spacing consistency
